## IMAGE GENERATION + TEXT FRAMEWORK ALIGNMENT

### **Current Misalignment:**

**Text System:**
- Topic: "Emergency AC repair"
- Selects: PAS framework
- Output: "Problem ‚Üí Agitate ‚Üí Solution"

**Image System:**
- Brand color: #FF5733 (red)
- Detects: Energetic personality
- Output: Vibrant, bold lighting

**Problem:** Text says "urgent problem" but image says "energetic excitement" - MISMATCH!

---

### **Proposed Integration:**

**Framework ‚Üí Image Mood Mapping:**

```typescript
const frameworkImageMoods = {
  'pas': {
    // Pain-focused ‚Üí serious, urgent
    moodOverride: 'urgent and serious',
    lightingStyle: 'dramatic with contrast',
    colorIntensity: 'high saturation for urgency',
    composition: 'close-up, intense focus'
  },
  
  'aida': {
    // Awareness-building ‚Üí inviting, approachable
    moodOverride: 'welcoming and clear',
    lightingStyle: 'bright, even lighting',
    colorIntensity: 'balanced, not overwhelming',
    composition: 'open, spacious'
  },
  
  'bab': {
    // Transformation ‚Üí before/after feel
    moodOverride: 'hopeful and aspirational',
    lightingStyle: 'uplifting, warm glow',
    colorIntensity: 'optimistic saturation',
    composition: 'progress, movement'
  },
  
  '4ps': {
    // Trust/proof ‚Üí professional, credible
    moodOverride: 'trustworthy and premium',
    lightingStyle: 'professional studio quality',
    colorIntensity: 'sophisticated, refined',
    composition: 'polished, authoritative'
  }
};
```

**Implementation:**
When framework selector chooses PAS, image generator should:
1. Keep brand colors
2. But apply PAS-appropriate mood (urgent, serious)
3. Override personality if conflict (energetic ‚Üí serious for PAS)
